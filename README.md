# user-management-react

  

Simple User Management System - Client Side with ReactJS and Redux

## Usage  

``` bash
# install dependencies
$ npm install
# serve with hot reload at localhost:3000
$ npm start
# serve with hot reload and Sass watch at localhost:7000
$ npm run dev
# production build
$ npm run build

```
